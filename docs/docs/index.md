# IP 扫盲：别™再被割韭菜

:::info 前言

**关于翻出去（VPN）**

在中国使用VPN本身就处于法律的灰色地带。这是一个现实，也是背景。

但需要明确的是：

- **VPN只是工具** —— 它本身没有好坏，就像刀子既能切菜也能伤人。关键看怎么用。

- **用途决定性质** —— 用VPN看个被和谐的视频、读个被墙的文章、使用个被屏蔽的服务，这是一回事。码农不想更换镜像，直接用VPN访问GitHub、Stack Overflow、npm源，这也是一回事。但用VPN去欺诈、骚扰、传播不实信息，又是另一回事。

- **不要被忽悠** —— 市面上很多人鼓吹「家宽IP」能做各种事（爆流量、骗系统、隐身做坏事），本质目的就是**赚钱**。这篇文章的作用是让你看清真相，别被这帮人骗。

这篇文章的核心很简单：**了解IP的真实知识，不被商人的谎言蒙蔽**。同时希望你明白——工具是中立的，但使用者要有底线。

:::

---

## 一、什么是家宽

家宽就是分配给普通家庭上网的 IP，whois 里 usage type 写着 **residential** 的才是。

就这么简单。下面开始打假。

---

## 二、忽悠话术大赏

### 1. "我们是 NTT / Cogent / GTT 家宽！"

放你妈的屁。

NTT、Cogent、GTT 是**骨干网运营商**，是修高速公路的，不是给你家装宽带的。

你上网经过 NTT 的线路，就跟你开车上了京沪高速一样——高速公路公司会给你发户口本吗？

这些商家就是赌你不懂，看到几个洋文缩写就觉得牛逼。骨干网和 IP 类型有几把关系？

### 2. "ping0 显示 Residential！"

ping0 的数据，**给钱就能改**。

你花点钱找他们"更正数据"，明天你的机房 IP 也能变成"家宽"。这种网站的数据你也信？

更搞笑的是 ping0 的风控值——它怎么算的？很大程度上是看"多少人用这个 IP 来访问 ping0"。

正常人谁没事查自己 IP？去查的全是机场用户在测节点啊！用的人越多，风控值越高。这他妈是什么狗屁逻辑？

所以那些热门节点风控值高，不是因为 IP 脏，是因为傻逼太多都跑去测。

### 3. "双 ISP！原生 IP！"

ping0 给 NTT 这种骨干网显示"双 ISP"，你动脑子想想：

NTT 有给日本家庭装宽带吗？ 没有。它是批发商。

就像中国移动是运营商，但华为不是——华为造设备的，不给你家拉网线。

这种"双 ISP"标签就是 ping0 瞎几把标的，国内商家拿来忽悠你。

### 4. "IP 纯净度 98%！风险值只有 2！"

纯净度是个**伪概念**。

哪个国际标准组织定义过"IP 纯净度"？IEEE？IETF？**没有**。

这些数字全是各平台自己编的，没有任何统一标准。今天这个网站给你 98 分，明天那个网站给你 30 分，你信谁的？

**都别信，因为这东西本来就不存在**。

### 5. "美国/新加坡/德国家宽，速度快！"

我直接笑出声。

你知道这些国家对家宽管得多严吗？

- 需要本地身份证明
- 需要实际居住地址
- 运营商严格限制 IP 用途
- **根本不可能批发给中国机场**

美国真正的家宽运营商：Comcast、AT&T、Verizon、Spectrum。

你买的"美国家宽"是这些运营商的 IP 吗？去 whois 查一下。**不是？那就是假的**。

市面上的所谓"家宽"，要么是技术伪造的，要么是灰产盗用的，要么干脆就是印度人和中国人联手骗你钱的。

你以为你捡到便宜了，其实你就是那个便宜。

## 三、延迟和速度——两码事，懂？

| 概念 | 意思 |
|------|------|
| 延迟 | 数据包从 A 到 B 要多久 |
| 速度 | 一秒能传多少数据 |

**亚太 30ms 拥堵成狗，可能还不如 200ms 的美国线路快。**

延迟低但带宽堵死了，你的数据包就在那排队等着。美国线路虽然远，但路宽，反而跑得快。

### "200ms 好慢啊呜呜呜"

慢你妈。

200ms = 0.2 秒。你眨个眼都要 0.3 秒。

你用来干啥？刷 TikTok？看 YouTube？这些全是缓冲的，延迟影响个锤子。

真正需要低延迟的：打 CS、打 LOL、视频会议、高频交易。

你一个看视频的，天天纠结 200ms 还是 50ms，纯属给自己找不痛快。

## 四、Shadowrocket 测速的坑

打开 Shadowrocket，TCPing 一测，哇，美国节点 20ms！好快！

**快你个头**。

这 20ms 是到**国内中转入口**的延迟，不是到美国的。

你的数据走的是：
```
你 → 国内入口（20ms）→ 美国落地（+150ms）
```

TCPing 只测了第一段。

**20ms 到美国？你他妈基本的初中物理常识都没有**

### 正确测法

| 类型 | 方法 | 原因 |
|------|------|------|
| 中转机场 | URL Connect | 测完整链路 |
| 直连机场（Hy2/VLESS） | ICMP Ping | Connect 会算 TLS 握手，虚高 |

## 五、真正该看什么

### ip2location

👉 [https://www.ip2location.com/demo](https://www.ip2location.com/demo)

**为什么看这个？**

- 改不了 —— 不是你给钱就能改的
- 大厂在用 —— Netflix 等风控系统的数据源
- 要企业资质才能申请更正 —— 不是阿猫阿狗都能去改

**看 Usage Type：**

- **DCH** = 机房
- **ISP** = 普通 ISP  
- **Residential** = 真家宽

### scamalytics

👉 [https://scamalytics.com/](https://scamalytics.com/)

查 IP 有没有被标记 spam/fraud。

**IP 好坏只有两种状态：被标记了，和没被标记。**

没有什么纯净度 98% 还是 95%，没被标记就是好的，被标记了就是脏的。**二元的，懂？**

## 六、关于 TikTok

"用家宽 IP 做 TK 能爆流量！"

**能爆你妈**。

TikTok 算法看的是：

- 完播率
- 互动率
- 用户停留

**不是看你 IP 是不是家宽**。

你内容是垃圾，用总统家的 WiFi 也是零播放。你内容牛逼，用网吧机房 IP 照样百万播放。

**别把自己内容不行的锅甩给 IP**。
